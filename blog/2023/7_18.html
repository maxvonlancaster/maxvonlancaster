Ideas for student js tasks

18th of July, 2023

Ideas for student js tasks for the course "Modern web-tech development" - beginner level

  • Generate a random password
    Description: Write a JavaScript function that generates a random password of a specified length, using a combination of letters, numbers, and special characters.
  • Calculate the area of a circle
    Description: Write a JavaScript function that takes the radius of a circle as input and calculates its area, returning the result.
  • Remove duplicate elements from an array
    Description: Write a JavaScript function that takes an array as input and removes any duplicate elements, returning a new array without duplicates.
  • Count the number of vowels in a string
    Description: Write a JavaScript function that takes a string as input and counts the number of vowels (a, e, i, o, u) in the string, returning the count.
  • Capitalize the first letter of each word in a string
    Description: Write a JavaScript function that takes a string as input and capitalizes the first letter of each word, returning the modified string.
  • Find the longest word in a string
    Description: Write a JavaScript function that takes a string as input and finds the longest word in the string, returning the longest word.
  • Calculate the power of a number
    Description: Write a JavaScript function that takes a base number and an exponent as input and calculates the result of raising the base to the exponent, returning the result.
  • Check if a string is a valid email address
    Description: Write a JavaScript function that takes a string as input and checks if it is a valid email address, returning a boolean.
  • Reverse the order of words in a string
    Description: Write a JavaScript function that takes a string as input and reverses the order of words in the string, returning the modified string.
  • Find the intersection of two arrays
    Description: Write a JavaScript function that takes two arrays as input and finds the intersection (common elements) between them, returning a new array with the common elements.
  • Find the most frequent element in an array
    Description: Write a JavaScript function that takes an array of elements as input and finds the most frequent element in the array, returning the element.

Middle level

  • Validate a credit card number
    Description: Write a JavaScript function that takes a credit card number as input and validates it based on the Luhn algorithm, returning a boolean indicating whether the number is valid or not.
  • Find the second smallest number in an array
    Description: Write a JavaScript function that takes an array of numbers as input and finds the second smallest number in the array, returning the result.
  • Sort an array of objects by a property
    Description: Write a JavaScript function that takes an array of objects, each having a specific property, and sorts the array based on that property, returning the sorted array.
  • Implement a stack data structure
    Description: Write a JavaScript class that implements a stack data structure with methods like push, pop, and peek.
  • Implement a queue data structure
    Description: Write a JavaScript class that implements a queue data structure with methods like enqueue, dequeue, and peek.
  • Implement a binary search algorithm
    Description: Write a JavaScript function that performs a binary search on a sorted array to find a specific element, returning its index or -1 if not found.
  • Convert a number to Roman numerals
    Description: Write a JavaScript function that takes a positive integer as input and converts it to Roman numerals, returning the Roman numeral representation.
  • Check if a string is an anagram
    Description: Write a JavaScript function that takes two strings as input and checks if they are anagrams (contain the same characters in a different order), returning a boolean.
  • Find the closest pair of points in a two-dimensional plane
    Description: Write a JavaScript function that takes an array of points (x, y coordinates) as input and finds the pair of points that are closest to each other, returning the two points.

Advanced level

  • Sort an array using a custom sorting algorithm
    Description: Write a JavaScript function that takes an array of numbers as input and sorts it using a custom sorting algorithm (e.g., merge sort or quicksort).
  • Find all permutations of a string
    Description: Write a JavaScript function that takes a string as input and finds all possible permutations of the characters in the string, returning an array of permutations.
  • Implement a graph data structure
    Description: Write a JavaScript class that implements a graph data structure with methods like addVertex, addEdge, and findShortestPath.
  • Flatten a nested array
    Description: Write a JavaScript function that takes a nested array as input and flattens it into a single-level array, returning the flattened array.
  • Implement a depth-first search algorithm
    Description: Write a JavaScript function that performs a depth-first search on a graph and returns the traversal order of the nodes.
  • Implement a binary search tree
    Description: Write a JavaScript class that implements a binary search tree data structure with methods like insert, delete, and search.
  • Implement a Dijkstra's algorithm
    Description: Write a JavaScript function that finds the shortest path in a weighted graph using Dijkstra's algorithm, returning the shortest path from a given source to a target node.
  • Generate all possible subsets of an array
    Description: Write a JavaScript function that takes an array as input and generates all possible subsets of the array, returning an array of arrays representing the subsets.
  • Implement a radix sort algorithm
    Description: Write a JavaScript function that takes an array of numbers as input and sorts it using radix sort, a non-comparative integer sorting algorithm.
  • Implement a linked list data structure
    Description: Write a JavaScript class that implements a linked list data structure with methods like insert, delete, and search.
  • Find the kth smallest element in an array
    Description: Write a JavaScript function that takes an array of numbers and an integer k as input and finds the kth smallest element in the array, returning the element.
  • Perform matrix multiplication
    Description: Write a JavaScript function that takes two matrices as input and performs matrix multiplication, returning the resulting matrix.

HTML tasks, that should go with it

  • Build a responsive navbar with dropdown menus
    Description: Create a responsive navigation bar with dropdown menus using HTML and CSS. Ensure that the navbar collapses into a hamburger menu on smaller screens.
  • Design a multi-step form
    Description: Create a multi-step form using HTML and CSS, where each step is displayed on a separate page or section. Implement navigation between steps and form validation.
  • Create a slideshow with automated transitions
    Description: Design a slideshow using HTML, CSS, and JavaScript, where images or content automatically transition at specified intervals. Include controls to manually navigate through the slideshow.
  • Implement a responsive timeline
    Description: Build a responsive timeline using HTML and CSS to display a chronological series of events or milestones. Each event should include a title, date, and
    Description.
  • Develop a pricing table
    Description: Design a pricing table using HTML and CSS to showcase different pricing plans or packages. Include features, prices, and call-to-action buttons for each plan.
  • Build a responsive grid gallery with filtering
    Description: Create a responsive grid gallery using HTML, CSS, and JavaScript. Allow users to filter the gallery items based on categories or tags using interactive buttons or dropdowns.
  • Design a newsletter signup form with validation
    Description: Create a newsletter signup form using HTML and CSS, and implement form validation to ensure that the entered email address is in a valid format.
  • Create an animated progress bar
    Description: Build an animated progress bar using HTML, CSS, and JavaScript. The progress bar should animate as the user scrolls down the page, reflecting their progress.
  • Develop a responsive accordion
    Description: Design a responsive accordion using HTML, CSS, and JavaScript. The accordion should expand and collapse sections of content when clicked, allowing users to view additional details.
  • Implement a sticky header
    Description: Create a sticky header using HTML, CSS, and JavaScript that remains fixed at the top of the page as the user scrolls. The header should change appearance upon scrolling.
  • Embed a video
    Description: Embed a video into an HTML document using the < video > tag and specify the video source, dimensions, and optional controls.
  • Implement a responsive image gallery
    Description: Build a responsive image gallery using HTML and CSS, where multiple images are displayed in a grid layout and adjust their size based on the screen size.
  • Create a contact page with Google Maps integration
    Description: Design a contact page using HTML and CSS, including a form for users to submit inquiries and integrate Google Maps to display the location.
